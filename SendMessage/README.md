## 简介
- 这是一个基于`QTcpSocket`多线程，实现的客户端和服务器端进行互相通信的小实验
- 支持客户端断开重连的功能
- 使用方式就是构建项目，然后发送信息，客户端和服务器端也都会显示对应的信息
