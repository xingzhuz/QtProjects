## 简介



- 这是基于`QTcpSocket`和多线程，实现的客户端发送文件到服务器，服务器接收并保存的小实验
- 客户端选择本地的文件进行发送，服务器端接收文件，并使用客户端发来的文件名存储在特定目录`resources`下
- 注意`resources` 的存放位置，当前程序是构建目录的上一级目录，不同到源码中修改即可
- 也支持断线重连功能
